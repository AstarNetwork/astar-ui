# Astar-UI

## Project setup

### Install
```
yarn install
```

### Run Demo page
```
yarn dev
```

### Build Library
```
yarn build:lib
```

### Build Docs
```
yarn build:doc
```

## Usage Of Components Built
**As Component**
```
  import { Header } from 'astar-ui';
  ...
  components:{
    Header
  }
```

**As Plugin**
```
  import astarUI from 'astar-ui';
    ...
  app.use(astarUI)
```
